# gulp

安装`gulp`、`bower`最好安装到全局上，在运行以下命令前，`node`环境也需要配置全局变量`NODE-PATH`，[点击此处](https://github.com/tangJunGit/blog/tree/master/nodejs#globle)

```
$ npm install gulp -g
$ npm install bower -g
```

gulp插件：

`gulp`,
`gulp-autoprefixer`,
`gulp-cache`,
`gulp-clean`,
`gulp-concat`,
`gulp-connect`,
`gulp-imagemin`,
`gulp-jshint`,
`gulp-less`,
`gulp-load-plugins`,
`gulp-minify-css`,
`gulp-rename`,
`gulp-uglify`,
`browser-sync`,
`require-dir`

**1.安装依赖包**
```
$ npm install
$ bower install
```
**2.启动服务**
```
$ gulp server
```
