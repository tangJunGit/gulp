# gulp+bower

安装`gulp`、`bower`最好安装到全局上，`node`环境也需要配置全局变量`NODE-PATH`，[`node`全局配置](https://github.com/tangJunGit/Blog/tree/master/NodeJS#globle)

```bash
npm install gulp -g
npm install bower -g
```

gulp插件：

`gulp`,
`gulp-autoprefixer`,
`gulp-cache`,
`gulp-clean`,
`gulp-concat`,
`gulp-connect`,
`gulp-imagemin`,
`gulp-jshint`,
`gulp-less`,
`gulp-load-plugins`,
`gulp-minify-css`,
`gulp-rename`,
`gulp-uglify`,
`browser-sync`,
`require-dir`
